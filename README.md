# f2e-solution

To run these solutions, please make sure that the Node environment has been installed.

## Check the Node environment:
```console
node -v
```

## Install Node:
[Node.js](https://nodejs.org/)

## How to use it ?
```console
git clone https://github.com/ckckckc/f2e-solution.git
cd f2e-solution
node main.js
```
